Sistem Pemantauan Parkir

Proyek ini bertujuan untuk mengembangkan sistem monitoring tempat parkir di lingkungan kampus Universitas X menggunakan aplikasi berbasis Java yang terhubung dengan database MySQL. Sistem ini dikembangkan sebagai solusi atas permasalahan keterbatasan area parkir yang sering penuh serta sulitnya petugas memantau ketersediaan tempat parkir secara real-time. Melalui aplikasi ini, pengguna atau petugas dapat melihat informasi tempat parkir berdasarkan lokasi, status (tersedia/ditempati), dan jenis kendaraan. Data ditampilkan dalam bentuk tabel interaktif dengan fitur CRUD seperti tambah, edit, dan hapus data. Proses bisnis utama meliputi pendaftaran tempat parkir oleh petugas, pembaruan status secara berkala, pemeriksaan ketersediaan oleh pengguna, dan pembuatan laporan penggunaan parkir. Selain memberikan solusi praktis, proyek ini juga menjadi latihan dalam merancang model data, membuat koneksi database, dan menampilkan data dinamis melalui antarmuka Java.

Sistem Pemantauan Ketersediaan Ruangan

Proyek ini bertujuan untuk membuat sistem pemantauan ketersediaan ruangan menggunakan aplikasi Java yang terhubung dengan database MySQL. Sistem ini dikembangkan sebagai solusi atas permasalahan pengelolaan ruangan yang kurang efisien di lingkungan kantor. Melalui aplikasi ini, pengguna dapat melihat status ruangan (tersedia atau ditempati), lengkap dengan informasi kapasitas dan lokasi. Proses bisnis yang didukung meliputi pendaftaran ruangan oleh admin, pembaruan status secara real-time, pemesanan ruangan, dan penyajian laporan penggunaan ruangan. Proyek ini juga menjadi latihan dalam merancang model data, membuat koneksi database, serta menampilkan data ke antarmuka Java secara dinamis.

Aplikasi Data Mahasiswa Berbasis Java (MVC + Bootstrap)

Proyek ini bertujuan untuk membangun aplikasi web sederhana berbasis Java yang digunakan untuk mengelola data mahasiswa. Aplikasi ini menampilkan daftar mahasiswa dalam bentuk tabel yang berisi NIM, nama, dan program studi. Sistem ini juga dilengkapi fitur pencarian data, penambahan mahasiswa baru, pengeditan data, dan penghapusan entri. Dengan adanya aplikasi ini, proses pengelolaan data mahasiswa menjadi lebih praktis, efisien, dan sistematis. Proyek dikembangkan sebagai bagian dari modul praktikum yang memperkenalkan konsep arsitektur MVC (Model–View–Controller) dalam pengembangan aplikasi web Java. Selain itu, framework Bootstrap 5 diterapkan untuk memperindah tampilan antarmuka pengguna (UI) dan mempercepat proses styling. Melalui proyek ini, mahasiswa berlatih membangun aplikasi yang terstruktur, fungsional, dan user-friendly, serta memahami koneksi antara aplikasi Java dan database MySQL melalui JDBC.

Aplikasi Data Karyawan

Aplikasi Data Karyawan adalah aplikasi desktop berbasis Java dengan antarmuka GUI (Swing) yang dirancang untuk membantu pengelolaan data karyawan secara praktis dan efisien. Aplikasi ini memungkinkan pengguna untuk melakukan proses CRUD (Create, Read, Update, Delete) terhadap data karyawan yang mencakup NIK, nama, jabatan, alamat, email, dan nomor telepon. Seluruh data yang dimasukkan ditampilkan dalam bentuk tabel secara real-time, sehingga memudahkan proses pemantauan. Fitur-fitur seperti tombol Tambah, Simpan, Edit, Hapus, Batal, dan Keluar disediakan untuk menunjang kemudahan penggunaan. Proyek ini sekaligus menjadi latihan dalam membangun aplikasi Java Swing, mengatur alur input data, dan mengelola logika program berbasis event.
